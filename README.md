# 🎓 Smart Campus Management System

A comprehensive web-based **Smart Campus Management System** developed using **PHP, MySQL, HTML, CSS, JavaScript, Bootstrap, and PHPMailer**. The system provides a centralized platform for managing academic activities, student records, faculty operations, assignments, attendance, announcements, events, marks, leave requests, and user profiles.

---

# 📌 Project Overview

The Smart Campus Management System is designed to automate and simplify campus administration by providing separate dashboards and functionalities for:

* Administrator
* Faculty
* Students

The system ensures efficient communication and management of academic data while maintaining role-based access control and security.

# ⚙️ Installation Guide

## Step 1

Install:

* XAMPP
* PHP 8+
* MySQL

---

## Step 2

Clone Repository

```bash
git clone https://github.com/PrawinKumar514/ApexPlanet-Internship-task-5.git
```

---

## Step 3

Move Project

```text
C:\xampp\htdocs\
```

---

## Step 4

Start Services

* Apache
* MySQL

---

## Step 5

Create Database

```sql
CREATE DATABASE smart_campus;
```

---

## Step 6

Import SQL File

Import the database file into phpMyAdmin.

---

## Step 7

Configure Database

Update:

```php
config/database.php
```

with your database credentials.

---

## Step 8

Run Project

```text
http://localhost/smart-campus/
```
